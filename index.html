01 — Langage

Java

Write Once, Run Anywhere — depuis 1995

Qu'est-ce que Java ?

Java est un langage de programmation orienté objet, statiquement typé, créé par James Gosling chez Sun Microsystems en 1995. Sa philosophie fondatrice — "Write Once, Run Anywhere" — repose sur la Java Virtual Machine (JVM), qui permet à un même bytecode compilé de s'exécuter sur n'importe quelle plateforme sans recompilation.

Aujourd'hui maintenu par Oracle, Java reste l'un des langages les plus utilisés au monde, présent dans les applications d'entreprise, Android, les systèmes embarqués et les serveurs web à haute disponibilité.

Architecture & JVM

Le code source Java (.java) est compilé en bytecode (.class) par le compilateur javac. Ce bytecode est ensuite interprété et optimisé par la JVM via le JIT compiler (Just-In-Time), qui le traduit en instructions machine natives au moment de l'exécution. Ce mécanisme garantit performances et portabilité simultanées.

La JVM gère également la mémoire via un Garbage Collector automatique, libérant le développeur de la gestion manuelle des allocations — contrairement au C ou C++.

Exemple de code

// Exemple Java — Polymorphisme & Héritage
public abstract class Animal {
    protected String name;

    public Animal(String name) {
        this.name = name;
    }

    public abstract String speak();

    @Override
    public String toString() {
        return name + " dit : " + speak();
    }
}

public class Dog extends Animal {
    public Dog(String name) { super(name); }

    @Override
    public String speak() { return "Woof!"; }
}

public class Main {
    public static void main(String[] args) {
        List<Animal> animals = List.of(new Dog("Rex"));
        animals.forEach(System.out::println);
    }
}

02 — Système d'exploitation

Linux

Le noyau qui fait tourner le monde — depuis 1991

Qu'est-ce que Linux ?

Linux est un noyau (kernel) de système d'exploitation open-source créé par Linus Torvalds en 1991, alors étudiant à l'Université d'Helsinki. Inspiré de MINIX et UNIX, Linux a rapidement évolué grâce à une communauté mondiale de contributeurs pour devenir le système d'exploitation dominant dans les serveurs, les supercalculateurs, les smartphones (via Android) et l'embarqué.

Techniquement, "Linux" désigne uniquement le noyau. Les distributions complètes — Ubuntu, Debian, Arch, Fedora — combinent le noyau Linux avec des outils GNU, un gestionnaire de paquets et un environnement de bureau.

Architecture du noyau

Le noyau Linux est de type monolithique modulaire : il tourne en espace kernel et gère directement le matériel. Il est composé de sous-systèmes : gestion des processus (scheduler), système de fichiers virtuel (VFS), pile réseau TCP/IP, gestion mémoire (VMM), et pilotes de périphériques.

Sa modularité permet de charger et décharger des modules (.ko) à chaud sans redémarrage, via les commandes insmod, rmmod et modprobe. Le noyau est écrit à 97% en C et 3% en Assembly.

03 — Jeu vidéo

Minecraft

Le jeu le plus vendu de l'histoire — depuis 2009

Qu'est-ce que Minecraft ?

Minecraft est un jeu vidéo de type "sandbox" développé à l'origine par Markus "Notch" Persson en Java, et lancé en version alpha en 2009. Racheté par Microsoft en 2014 pour 2,5 milliards de dollars, il est aujourd'hui développé par Mojang Studios. Avec plus de 300 millions de copies vendues, c'est le jeu le plus vendu de tous les temps, tous supports confondus.

Le monde de Minecraft est généré procéduralement à partir d'un "seed" — une graine numérique. Il est composé de chunks de 16×16×384 blocs, formant un monde théoriquement infini (environ 60 millions de km² de surface explorable).

Technique & Moteur

Initialement entièrement écrit en Java, Minecraft Java Edition reste la version de référence pour le modding et les serveurs. La version Bedrock Edition (C++) est multiplateforme (mobile, consoles, Windows). Le moteur maison gère le rendu de chunks, l'éclairage dynamique, les entités, la physique des fluides et les pistons.

Depuis la version 1.17, le moteur de rendu a été partiellement réécrit. Iris Shaders et Sodium (mods communautaires) permettent d'activer des shaders GLSL avancés, transformant radicalement le rendu visuel du jeu.
